Skyhopper

SkyHopper is a DevOps automation tool for Managing your AWS Cloud. Written in Ruby on Rails. It uses the following technologies: Chef - for managing cookbooks and system automation. AWS SDK- for managing infrastructures and stacks. Vue.js - frontend Javascript framework Go lang - for push notifications Ruby on Rails - Backend Implementation of the project. Zabbix - for monitoring and health checks of your EC2 instances.

  • Company: Skyarch Networks INC.
  • Date: August 2015 - Present
  • Service: Web Development

BeckTaxi OMDA

The Beck Taxi OMDA is backend system on Google Apps Engine for order management and taxi dispatching for Beck Taxi. It integrates with three mobile apps being developed by the Cloud Sherpas mobility team: a new in-car Android app for taxi drivers, and iOS and Android versions of a customer mobile app for web orders.

  • Company: Cloud Sherpas INC.
  • Date: July 2015
  • Service: Web Development

Codex for All

This app lets users to collaborate in real time on Firepad. Each users join a workspace and can watch each other make changes in real time — if user A types something new, user B sees it appear on his screen in real time on specific url. It has simple design, does have line numbering, settings for user; overall its easy to use.
